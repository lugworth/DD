// how to use

Copy tailwind.config.js into your project, or paste its theme.extend block. Tokens map to standard Tailwind utilities: bg-coral, text-sky, border-teal, font-display. Radius is globally reset to 0. Motif classes (.chk, .tex.flower) live in the @layer utilities block.
