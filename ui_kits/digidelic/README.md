# digidelic UI Kit

## Overview
Interactive recreation of the digidelic interface aesthetic — mission control / network operations dashboard.

Built from brand description + design system tokens. No codebase was attached.

## Files
- `index.html` — Main interactive app (node grid, terminal, detail panels)

## Screens / States
1. **Grid View** — Node map showing all sector nodes with status
2. **Node Detail** — Selected node inspector with data readout
3. **Terminal** — System terminal with command input

## Components Used
- `<SideNav>` — Left navigation with sector list
- `<NodeGrid>` — Main content grid of data cards
- `<DetailPanel>` — Right-side node inspector
- `<TerminalPanel>` — Bottom terminal log
- `<TopBar>` — Header with system status

## Design Tokens
All tokens from `../../colors_and_type.css` — hard corners, monospace UI, yellow accent.

## Notes
- No codebase was provided; this is synthesized from brand aesthetic
- Fonts: Barlow Condensed (headings) + Space Mono (UI) via Google Fonts
