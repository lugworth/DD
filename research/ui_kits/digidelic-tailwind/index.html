// how to use

Copy tailwind.config.js into your project, or paste its theme.extend block. Tokens map to standard Tailwind utilities: bg-coral, text-sky, border-blood, font-display. Radius is globally reset to 0. Color-blocking is the core move: loud states become solid color blocks with a black data inset; calm states stay black void with a single color rule. Motif + texture classes (.chk, .tex.night, .swirl) live in the @layer utilities block.
